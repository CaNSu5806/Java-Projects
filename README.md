## Current Project

### Word Separator & Text Analyzer

A simple Java console application that analyzes a user-provided text.

#### Features
- Counts:
  - Vowels
  - Consonants
  - Spaces
- Finds the most frequently used letter
- Finds the most frequently used word(s)
- Uses arrays and `ArrayList`
- Accepts user input through the console

#### Concepts Practiced
- Variables and data types
- Loops (`for`)
- Conditional statements (`if-else`)
- Arrays
- `ArrayList`
- String manipulation
- User input with `Scanner`
- Basic frequency analysis
